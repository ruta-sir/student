## Example Endpoints

### `GET /chatbot/info`

Returns basic information about the Gemini Chatbot.

```java
@GetMapping("/chatbot/info")
public ResponseEntity<String> getInfo() {
    return ResponseEntity.ok("Gemini Chatbot v1.0 - Ready to assist!");
}
```
### `POST /chatbot/message`

Accepts a user message and returns a chatbot response.

```java
@PostMapping("/chatbot/message")
public ResponseEntity<String> handleMessage(@RequestBody String userMessage) {
    String reply = generateReply(userMessage);
    return ResponseEntity.ok(reply);
}
```


## Postman Testing Highlights
Using Postman, I was able to:
- Send GET requests to verify static responses.
- Submit POST requests with JSON bodies to simulate user input.
- Inspect headers, status codes, and response times.
- Debug unexpected behavior and refine endpoint logic.
